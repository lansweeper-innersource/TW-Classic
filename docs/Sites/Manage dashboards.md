<!-- # Manage dashboards -->
Lansweeper Sites' dashboards provide an overview of crucial information about your environment.

Dashboards are made up of a variety of widgets. Widgets visually represent the data retrieved from various views and [reports](/docs/introduction-to-the-report-builder).

Want to learn more about all the reports Lansweeper offers? Check out our [Reports Library](https://www.lansweeper.com/report/)!

On this page:

- View dashboards
- Create dashboards
- Configure widgets
- Share dashboards
- Update dashboard information

## View dashboards

To view your dashboards, go to your Lansweeper site, and select **Dashboard**. The following pre-configured dashboards are available:

- General overview
- Assets
- Inventory health
- Security
- Software
- OT overview
- Lifecycle

## Create dashboards

Create custom dashboards to easily view your desired information in the visual representation of your choice.

To create a new dashboard:

1. Go to **Dashboard > Create new board.**
2. Give your board a name, then select **Create board****.**
3. Select **Add new widget** ![+.png](/docs/.document360/assets/article_339/image_002.jpg).
4. Configure widgets as necessary.
5. Select **Save board** ![save.png](/docs/.document360/assets/article_339/image_003.jpg).

### Configure widgets

You can configure the following widgets once you've created or edited a dashboard.

For all widgets, select **Edit ![edit.png](/docs/.document360/assets/article_339/image_004.jpg)** to edit the widget's information. Select **Delete ![delete.png](/docs/.document360/assets/article_339/image_005.jpg)** to delete the widget.

Select and drag a widget to reorganize it on your Dashboard.

Select and hold the bottom-right corner of the widget to adjust its size.

#### iFrame

Embed content from another webpage within your board.

Enter a valid URL and select **Apply**.

#### RSS

Get the latest news from Lansweeper or subscribe to any other RSS feed.

1. Either:
   - For Lansweeper's RSS, select **Lansweeper news**.
   - For other RSS feeds, select **URL** and enter the RSS feed's URL.
2. Select **Snippet** and **Dates** if you want the additional information to display.
3. Select **Apply**.

#### Charts

Display data from a report or a view as one of the different chart types available.

The table columns display each view or report's name, type, source, category, subcategory, and description.

1. From the table, you can choose which type you want to display from the dropdown. By default, all types are available. You can also filter by Type:
   - **View**: Provides you with real-time data and is perfect for simpler queries.
   - **Report**: Can meet your more complex needs but must be manually updated.
2. Select the view or report you want, then **Next**.
3. Select the chart type from the dropdown, and choose to **Display chart legend** and **Display data in a table**. Select **Apply**.

#### Numbers

Get a real-time count from different sources, like views and reports.

The table columns display each view or report's name, type, source, category, subcategory, and description.

1. From the table, you can choose which type you want to display from the dropdown. By default, all types are available. You can also filter by:
   1. Type:
      - **View**: Provides you with real-time data and is perfect for simpler queries.
      - **Report**: Can meet your more complex needs but must be manually updated.
   2. Source:
      - **Built-In**: Views or reports built-in by Lansweeper.
      - **Custom**: Views or reports that you created.
2. Select the view or report you want, then **Apply**.

#### Report overview

Select a group of reports and list their results. Use this widget to keep track of your most critical reports.

Select **Hardware**, **Latest vulnerabilities**, or **Software**, then select **Apply**.

To create a new overview:

1. Select **Add new overview**.
2. Enter a name for your overview and select **Create Overview**.
3. Select **Add report** under High Priority, Important, or Informational. You can also select **Add new group** and name the group.
4. Select as many reports as you want to add, then select **Apply and save**.

#### Tables

Display data from a report or a view as a table.

The table columns display each view or report's name, type, source, category, subcategory, and description.

1. From the table, you can choose which type you want to display from the dropdown. By default, all types are available. You can also filter by:
   1. Type:
      - **View**: Provides you with real-time data and is perfect for simpler queries.
      - **Report**: Can meet your more complex needs but must be manually updated.
   2. Source:
      - **Built-In**: Views or reports built-in by Lansweeper.
      - **Custom**: Views or reports that you created.
2. Select the view or report you want, then **Next**.
3. Select **Apply**.

#### Scanning status

Check your inventory health by monitoring your installations and scan servers' status.

Either:

- Select **All installations and automatically the new ones** **> Apply**.
- Select **Some specific installations**. From there, select the installations you want to preview, then **Apply**.

#### Asset overview

Get an overview of your inventory through its categories, such as groups, types, domains, workgroups, etc.

Select which assets you want to display from the list, then **Apply**.

#### Manufacturer overview

Get an overview of your inventory filtered by manufacturers and models.

Select **Model** or **Asset type** if you want to display those as well. Then select **Apply**.

#### Windows overview

Get an overview of your inventory filtered by Windows assets.

Select **Operating system version** if you want to display that as well. Then select **Apply**.

## Share dashboards

1. Go to **Dashboard** and select the custom dashboard you'd like to share.
2. Select **Share ![share.png](/docs/.document360/assets/article_339/image_006.jpg)**.
3. Choose to share the board with:
   - **All accounts and automatically the new ones**.
   - **Some specific accounts**.
4. Select **Share board**.

## Update dashboard information

Widgets that display views are updated in real-time.

Widgets that display reports are updated when the report is run. However, you can also update your Dashboard's information immediately by running the reports from the Dashboard.

To update your Dashboard's information:

1. Go to **Dashboard** and select your desired report.
2. To update specific widgets, locate the widget and select ![run-reports.png](/docs/.document360/assets/article_339/image_007.jpg) **>** **Run report now**.
3. To update the entire board, select **Run all board reports** ![run-reports.png](/docs/.document360/assets/article_339/image_008.jpg).
