<!-- # Scan Windows computers targets -->
Scan Windows domain or workgroup computers by adding Windows computers as an agentless scan target.

To add a Windows computer scan target:

1. In your Lansweeper Cloud environment, navigate to **Scanning > Targets**.
2. Select **Add scanning target**.   
   ![Nils_0-1677840764400.png](/docs/.document360/assets/article_392/image_002.jpg)
3. Choose an available scan server to configure the target, select **Windows computer** from the list of target types, then select **Add target**.
4. Enter the target credentials:  
   - Target: a name for your scanning target. This is for personal reference only.
   - Domain (Netbios): your domaind's Netbois name. If you're unsure what this name is, see [Find your domains' DNS and NetBOIS names](/docs/finding-your-domains-dns-and-netbios-names "Find your domains' DNS and NetBOIS names").
   - Description: an optional description for your scanning target. This is for personal reference only.![Windows-Computer-Main-Information.png](/docs/.document360/assets/article_392/image_003.jpg)
5. Select a scanning schedule for the Windows computer target.
6. Select an available credential to execute the scanning target.![Windows-Map-Credentials.png](/docs/.document360/assets/article_392/image_004.jpg)
7. Select **Save and exit**, or **Save target** **![sophie_0-1677859532366.png](/docs/.document360/assets/article_392/image_005.jpg).**
8. In the pop-up, choose whether you want to scan the Windows computer target immediately.

You can view your new scanning target in **Scanning > Targets > All targets**.

The scan will run as scheduled. To scan your target immediately, select **Scan**.
