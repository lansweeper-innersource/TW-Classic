![TL;DR-Sweepy-Icon (1).png](/docs/.document360/assets/article_089/image_001.jpg) **Automate asset data syncing from Lansweeper to ServiceNow CMDB with Syncfish's CI Synchronizer, ensuring accurate, real-time updates for improved IT operations, compliance, and risk management.**

Integrate Lansweeper with Syncfish's CI Synchronizer (CI Sync) to automate data synchronization from IT, OT, and IoT assets into your ServiceNow CMDB. This integration ensures your ServiceNow CMDB always has accurate, up-to-date data, improving IT operations, compliance, and risk management.

To learn more about CI Synchronizer, check out [CI Synchronizer - The Ultimate CMDB Automation Solution](https://www.syncfish.com.au/products/engine/ci-synchronizer/?utm_source=LS&utm_medium=Web&utm_campaign=CE-LS-WEB&utm_term=CI+Synchronizer+Enterprise+Edition&utm_content=5).

## Key benefits

- **Automated data synchronization:** Lansweeper discovers asset data, and CI Sync maps and correlates the data to create precise CI relationships, streamlining workflows.
- **Rapid deployment:** CI Sync offers an easy setup and seamless integration with ServiceNow.
- **Enhanced ITSM workflows:** Combine asset discovery with real-time updates to maintain consistent and reliable IT processes.
- **Risk mitigation:** Reduce compliance, cyber, and operational risks with accurate asset data and real-time updates.

## Requirements

- **Azure AD integration:** Used for user authentication when scheduling sync jobs and reviewing logs.
- **On-premise Windows VM:** This VM hosts the CI Sync Agent for Lansweeper. It can be the same VM that runs your Lansweeper Scanning Service.
